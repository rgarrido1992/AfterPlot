# AfterPlot

**Track your favorite TV shows with precision and emotion.**

AfterPlot is a modern web application for tracking TV series, powered by analytical insights, community engagement, and a beautiful, responsive interface.

## Features

- 📊 **Advanced Analytics** - Track your viewing habits with detailed statistics
- 🎬 **Series Tracking** - Mark episodes as watched and follow series status
- 💭 **Emotional Feedback** - Record your feelings and reactions to each episode
- 🔍 **Smart Discovery** - Explore new shows by genre and trending content
- 📱 **Fully Responsive** - Beautiful experience on mobile, tablet, and desktop
- ⭐ **Community Ratings** - Rate episodes and vote for MVP characters

## Tech Stack

- **Frontend**: Next.js 15 + React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL (hosted on Railway)
- **External APIs**: TheMovieDB (TMDB), JustWatch

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app in development mode.

## Environment Variables

Create a `.env.local` file with:

```
DATABASE_URL=your_railway_postgresql_url
NEXT_PUBLIC_TMDB_API_KEY=your_tmdb_api_key
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```

## Deployment

This project is configured to deploy on Railway.
